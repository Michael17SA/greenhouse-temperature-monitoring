# Portfolio Note

## CV Bullet

Built a C# console-based greenhouse temperature monitoring system that stores sensor readings in a two-dimensional array and calculates average temperatures per section.

## LinkedIn Description

I developed a Greenhouse Temperature Monitoring System in C# to practise working with arrays, loops, input validation, and basic data analysis. The application allows a user to enter temperature readings for different greenhouse sections, view the stored readings, and calculate the average temperature for each section.
