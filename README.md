# Greenhouse Temperature Monitoring System

A C# console application that records temperature readings for multiple greenhouse sections and calculates the average temperature for each section.

## Overview

This project demonstrates the use of two-dimensional arrays, input validation, loops, and basic data analysis in C#. The user enters the number of greenhouse sections, the number of readings per section, and the temperature values collected from each section.

## Features

- Captures temperature readings for multiple greenhouse sections
- Stores readings in a two-dimensional array
- Validates numeric user input
- Displays all recorded sensor readings
- Calculates and displays the average temperature per section

## Technologies Used

- C#
- .NET 9
- Console application

## How to Run

```bash
dotnet run
```

## Example Use Case

This application can be used as a beginner-friendly simulation of how greenhouse temperature data may be captured and summarized for monitoring purposes.
